# simple-fronted-apm

简易的web端性能监控系统  
* 内有行为监控、页面性能监控和异常监控； 
* 实现数据采集与业务系统完全解耦；
* 解决数据上报丢包、跨域等问题；
* 使用简单、易上手、以及极少的第三方依赖；


<br>
